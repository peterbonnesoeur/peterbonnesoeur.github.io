# Hello World

Welcome to my new corner of the internet. If you're reading this, you've found the blog section of my freshly rebuilt portfolio.

## The Stack

This site is built with **Astro** + **Tailwind CSS v4** + **MDX**, deployed for free on **GitHub Pages**. It ships zero JavaScript by default — interactive bits only load where needed.

## Why the Trail Theme?

I'm training for ultra trail running, and the aesthetic of topographic maps felt like a perfect match for a portfolio that tells a career story. The elevation profile timeline on the homepage is my favorite piece — every career peak is literally a peak on the chart.

## Easter Eggs

I've hidden a few things around the site. Here are some hints:

- Try pressing the **backtick** key anywhere
- There's a classic video game cheat code that works here
- Click my photo a few times
- Navigate to a page that doesn't exist
- Find the hidden summit markers (there are 5)

Happy exploring!
